This is a short demo of a custom cutscene I created using models, textures, and audio from *Halo: Combat Evolved*. All of this is rendered in real-time on a custom OpenGL-based graphics engine that supports features like **inverse kinematics**, **dynamic shadows**, **particle effects**, **texture mapping**, and more.

This was originally created for [CS 488: Intro to Computer Graphics](https://student.cs.uwaterloo.ca/~cs488/Winter2026/index.html) at the University of Waterloo in Winter 2026, my last term. Many all-nighters were pulled on this project.
